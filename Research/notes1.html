18-02-14: Adjoint and Classifying Spaces

In general, we define the classiyfing space K(G,n) (also known as an Eilenberg-MacLane space) to be a space X such that πn(X)=G and for k≠n, πk(X)=0.

Note: in my notation, I will simply write this as π∗(X)=Gδn

It is worth mentioning here that there are nice Serre spectral sequences for this family of fibrations:

K(Z,n−1)→{pt}→K(Z,n)

By examining an appropriate spectral sequence, we were able to find that H∗(RP∞)=Z2δ1, which makes RP∞ an geometric model of the classifying space K(Z2,1).

Recall that CP∞ is defined as the limit of the sequence of inclusions

CP1⊂CP2⊂CP3⊂…
together with the weak limit topology.

There are a handful of easily recognizable geometric models for a few other types of classifying spaces.

Using spectral sequences, we also found that K(Z,3) was a space that, although simple from the point of view of homotopy, had a more complicated structure in homology. It was a number of odd properties- it has torsion in infinitely many dimensions, doesn’t satisfy Poincare duality (even in a truncated sense).

If we know H∗(CP∞), which is the easiest case, we can then use the inclusion CPn→iCP∞ (as a cellular map) to induce

H∗(CPn)−→i∗H∗(CP∞)β↦β

which is a actually a ring homomorphism instead of just a group homomorphism. This presents a good argument for the use of cohomology, due to its extra ring structure.

This is an isomorphism on low-dimensional (co)homology, which reflects the idea encapsulated in the weak limit that these should be approximately equal for large enough n.

This is indicative of a general principle: if X is a CW complex and Xn is its n-skeleton, then the inclusion Xn→iX induces an isomorphism Hk(Xn)≅Hk(X) for k<n. (Note that this may or may not be an isomorphism for k=n.)

In particular, it is again a ring homomorphism, and so carries true relations/equations to true relations/equations.

Dually, homology does have some type of ring structure, however it is slightly unnatural and onerous to define and use. There is a natural coproduct on H∗(X) for any space X, which has a “one in, two out” type and takes this form:

H∗(X)−→ΔH∗(X)×H∗(X)a↦∑a′⊗a′′

This coproduct satisfies a form of coassociativity, i.e. if we have

Δ(a)=∑bi⊗ci(Δ⊗1)Δ(a)=∑i,j(dij⊗eij)⊗ci(1⊗Δ)Δ(a)=∑i,jbi⊗(fik⊗gik)

then the “structure coefficients” agree, i.e. we have bi=∑j(dij⊗eij) and ci=∑k(fik⊗gik).

In other words, just note that each element on the right hand side of these equations is an element of H⊗3∗, and so coassociativity simply requires that they are the same element of this space.

We can specialize by looking at the case where V is a vector space, with a coproduct V−→ΔV⊗V. Then pick a basis {ei}i∈I, and write

Δ(ei)=∑j,kΔj,ki(ej⊗ek)

where Δj,ki∈k, the ground field of V. Then coassociativity requires that we have

∑j,k,l,mΔj,kiΔl,mj(el⊗em⊗ek)=∑j,kΔj,kiej⊗Δp,qk(ep⊗eq)

or in other words, that

∑jΔj,kiΔl,mj=∑iΔl,riΔm,kr∀k,l,m

It is worth noting that there is also a version of the universal coefficient theorem for homology, which comes in the form

0→Ext(Hn−1(X,Z),Z)→Hn(X,Z)→Hom(Hn(X,Z),Z)→0

One question that comes up here is whether or not there is a sense in which Ext and Hom are “duals” of each other. In some way, this is case, using the “Frobenius duality” of ⋅⊗R and Hom(⋅,S).

Aside: Frobenius duality occurs in algebras A over some field k possessing a nondegenerate bilinear form A×A→σk satisfying σ(ab,c)=σ(a,bc). Such a \sigma$ is called a Frobenius norm. A simple example is the trace of a matrix, another example is any Hopf algebra.

This kind of duality comes in the form of something like

Hom(M⊗N,P)=Hom(M,Homin(N,P))

where Homin is an “internal hom”, which is actually an object in the category whose underlying set is the usual Hom. One might also call this “map”, and denote it [N,P], then the above statement translates to the condition that if N,P∈C for some category, then Homin(N,P)=[N,P]∈C is also an object in the same category. (This might also be denoted Hom.)

For an analogy, let C=Top, and HomTop(X,Y) be the set of continuous maps from X to Y. Then notice that we can put a topology on this space, say T, so define Map(X,Y)=(HomTop(X,Y),T), which is in fact an object in Top. This becomes the aforementioned “internal hom”.

Then, the previous adjunction becomes

HomTop(X×Y,Z)=HomTop(X,Map(X,Y))(∈Set)

More generally, consider what happens in categories of R modules, where R is generally non-commutative. We can then take objects like MR∈mod-R and RNS∈R-mod-S. We can then form the tensor product MR⊗RRNS, and the adjunction becomes

Hommod-S(MR⊗RRNS,PS)=Hommod-R(MR,Hommod-S(RNS,PS))(∈Ab)

Again, in the second argument of the right-hand side, we identify this as an internal hom - this works because the object Hommod-S(RNS,PS) actually becomes a right R-module by precomposition.

In some ways, this resembles the kind of adjunction that occurs in an inner product space - for example, given a matrix A, it may have an “adjoint” matrix A∗ that satisfies

⟨,Av⟩w=⟨,w⟩A∗v

and so we can think of Hom like a Hermitian inner product of this form, which is contravariant (re: conjugate) in the first argument. Note that the choice of which argument is contrvariant varies! In Physics, the second argument is often conjugate-linear, while the first is linear.

In this framework, we can now talk about pairs of adjoint functors C↔RLD between categories, which satisfy

HomC(LA,X)=HomD(A,RX)

for every A∈D,X∈C, plus a few more properties concerning how these act under natural transformations.

Then L is said to be left adjoint to R, and R is right adjoint to L, which is sometimes denoted L⊢R.

Example: Free and forgetful functors.

Work in Grp and Set, then let F be the free group functor and U by the forgetful functor. Then we have

HomGrp(F(S),G)≅HomSet(S,U(G))

On a different note, it is worth mentioning some of the fibrations to which a spectral sequence might apply. One that comes up is

Un−k×Uk→Un→GrC(n,k)

where GrC(n,k) is the set of k-planes in Cn. Here, it is worth noting that Un≃GLn(C) and On≃GLn(R).

From this, it can be concluded that GC(k,n)=UnUn−k×Uk, and further that if we take limn→∞ we obtain GrC(k,∞)=XUk, where X is some contractible space, and we thus find that GrC(k,∞)=BUk, the classifying space for Uk.

It can further be shown that there is another fibration

Uk→EUk→BUk

where EUk is a contractible space on which Uk acts and BUk is the above quotient. We can then find interesting structure here arising from the fact that H∗(GrC(k,∞))=H∗(BUk).
