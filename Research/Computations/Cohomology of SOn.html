1) Cohomology of SO3

A priori, we can use the fact that SO3 is diffeomorphic to RP3 to obtain

H∗(SO3)=H∗(RP3)=[Z,0,Z2,Z,0,0,⋯]

But supposing we didn’t know the cohomology of the total space, the Serre spectral sequence can be applied by using the fibration

S1→SO3→S2

and then the usual theorems give the formula (in cohomology)

Ep,q2=Hp(S2)⊗Hq(S1).

This yields the following E2 and E3 pages respectively (indexing columns with p and rows with q):

where the dots are zeros, circled entries have stabilized, and the trivial differentials are suppressed.

This allows us to conclude:

  • H0(SO3)=Z
  • H3(SO3)=Z

But so far, nothing about H1 or H2.

We know that d0,12∈Hom(Z,Z), so there are a few cases:

  1. d0,12(1)=0, so it’s the zero map, and we obtain this E2 page:

    Z 0 0 0
    Z 0 0 0

  2. d0,12=±1, so it’s an isomorphism, yielding

    0 0 Z 0
    Z 0 0 0

  3. d0,12(1)=±n for some n≠0,1, in which case there’s torsion:

    0 0 Z 0
    Z 0 Zn 0

In any case, at most one of the diagonals survive. So this leads to the following questions:

Question 1.1

Can this be pushed any farther without bringing in external information?

For example, a posterio we know that SO3≅RP3 which forces case (3) to hold, where n=2 and d0,12(1)=2. (Is there anything interesting about knowing this differential explicitly?)

Is there any more algebraic information here, or is this the point where one would turn to geometric information to continue?

Question 1.2

Supposing the cohomology is known, how do you write the ring/algebra structure?

By pushing generators around with cup products, it looks like a polynomial ring, but I’m not quite sure how to handle the fact that there are both Z and Z2 coefficients around.

One easy reduction is just to take Z2 coefficients, then I think H∗(RP3;Z2)≅Z2[x]/(x4) as algebras.

For α∈H∗(RP3;Z) though, it seems like it can be written as Z⊕Z2e2⊕Ze3 where ei is a generator in degree i. Is there a nice/clean way to express this as some polynomial ring Z[stuff1]/(stuff2), or maybe as some algebra Λ(stuff)?

Question 3.1

Do you think it’s possible to proceed from here using purely algebraic information? Is it tractable to try to dig into the spectral sequence and figure out some way to identify and compute these differentials, given that they’re 4 pages into the sequence?

Question 3.2

What actually is H∗(SO3), even just as a group? Is it actually possible to obtain it with this spectral sequence?

Hatcher seems to do some computation involving Bockstein homomorphisms on page 309 and comes up with:

H∗(SO5)≅F[x,y,z]/(2x,x4,y4,z2,xz,x3−y2)

where F is a “free graded commutative associative algebra over Z”. Is this the same thing as a DGA? And how exactly does one extract the list of cohomology groups from this representation?
