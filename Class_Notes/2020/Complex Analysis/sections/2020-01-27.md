Fix a connected domain $\Omega$ which is bounded with a piecewise $C^1$ boundary. 

Theorem (Green's):
Given $f, g \in C^1 \bar \Omega$, we have

\begin{align*}
\int_{\bd \Omega} f~dx + g~dy = \iint_{\Omega} \qty{ \dd{g}{x} - \dd{f}{y}  } ~dA \\
\int_{\bd \Omega} - f~dx + g~dy = \iint_{\Omega} \qty{ \dd{g}{x} + \dd{f}{y}  } ~dA \\
.\end{align*}
