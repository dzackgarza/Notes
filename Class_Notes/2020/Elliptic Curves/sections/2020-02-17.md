Theorem:
If $A/K$ is an abelian variety, $D\in \div(A)$, ample, $\ell(D) \geq 2$, $[D]$ symmetric, then there exists
a canonical height
$\hat h_{A, D}: A(K\sep) \to \RR$ 
such that

1. $\hat h_{A, D} = h_{A, D} + O(1)$
2. $\hat h_{A, D}$ is a *quadratic form* (defined last time).
